TEAM TASK MANAGER APPLICATION

---

## PROJECT OVERVIEW

This is a full-stack Task Manager web application that allows users to register, create tasks, and track them efficiently.

The application is built using Node.js, Express, and MongoDB, and is deployed online for public access.

---

## FEATURES

* User Signup
* Create Tasks
* View All Tasks
* Task Tracking (Status display)
* REST API integration

---

## HOW TO RUN LOCALLY

1. Clone the repository

2. Install dependencies:
   npm install

3. Create a .env file and add:
  MONGO_URL=mongodb+srv://shiva_reddy_010:shiva123@cluster0.dpqbwhn.mongodb.net/?appName=Cluster0

4. Start the server:
   node server.js

5. Open in browser:
   http://localhost:5000

---

## API ENDPOINTS

POST /signup

* Register a new user

POST /tasks

* Create a new task

GET /tasks

* Get all tasks
